# E2E 테스트 매트릭스

`tests/e2e/app-flows.spec.ts`에서 아래 핵심 시나리오를 자동 검증합니다.

## 공통/비로그인

- 회원가입 화면 진입
- 이름/전화번호 필수 필드 노출 확인
- 전화번호 숫자 입력 제한 확인
- 로그인 화면에서 비밀번호 찾기 페이지 이동

## 회원 플로우

- 로그인 성공
- `/my` 화면 진입 및 코치 피드백 영역 확인
- `/book` 화면 진입 및 정책 배너 확인
- 로그아웃

## 관리자 플로우

- 로그인 성공
- 관리자 대시보드 진입
- 탭별 접근 확인:
  - 시간 슬롯
  - 회원
  - 공지사항
  - 정책
  - 통계
- 회원 탭에서 코치 피드백 저장
- 로그아웃

## 수동 검증 권장 항목

외부 연동 특성상 아래 항목은 수동 검증이 안정적입니다.

- 비밀번호 재설정 메일 실제 수신/링크 동작
